Installation Instructions

  1. Tap "Get Device ID" button above to install the profile
    Location: Right on this page
  2. Install the profile in your device settings
    Location: Settings → General → VPN & Device Management
  3. Enable Developer Mode on your device
    Location: Settings → Privacy & Security → Developer Mode → Toggle On
  4. Restart your device when prompted
    Your device will automatically prompt you to restart
  5. Wait for developer authorization
    The developer will notify you when access is granted
  6. Return to this page and tap "Install App" button at the top
    Location: Top of this page

Frequently Asked Questions

What is a UDID and why do you need it?
A UDID (Unique Device Identifier) is a specific code that identifies your iOS device. We need this to register your device with Apple for testing purposes, as this app is not available on the App Store. This is a standard requirement for installing development apps on iOS devices.
Is it safe to install this profile and enable Developer Mode?
Yes, it's safe. The profile is only used to obtain your device's UDID. Developer Mode is an Apple feature that allows installation of apps from sources other than the App Store. You can disable Developer Mode at any time after installation if desired.
How long does the authorization process take?
Once you send your UDID to the developer, authorization typically takes 1-24 hours. You'll receive a notification from your coach when your device has been authorized and you can proceed with the installation.
Why do I need to restart my device?
Restarting is required by iOS when enabling Developer Mode. This is a security feature implemented by Apple to ensure you're intentionally enabling this mode. You'll only need to do this once.
Will this affect my device's warranty or security?
No, installing this profile and enabling Developer Mode won't affect your device's warranty or compromise its security. These are standard Apple features used by developers and beta testers. You can remove the profile and disable Developer Mode at any time.
